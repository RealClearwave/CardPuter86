CardPuter86 default DOS disk

This image is built from tools/dos/image_root.

Included tools:
- Svarog86/FreeDOS boot files and utilities.
- CP86TEST.COM: CardPuter86 hardware test.
- MININASM.COM: tiny 8086 assembler.
- PHANTOM.MID: user-provided MIDI sample.

Try:
  CP86TEST
  TE README.TXT
  MININASM

MIDI playback is not bundled yet until a redistributable PC speaker MIDI player is selected.
