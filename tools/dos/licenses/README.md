# DOS image third-party contents

The public CardPuter86 default image is built from `tools/dos/image_root`.
Only redistributable/user-provided files are placed here.

## Svarog86 / FreeDOS files

Source: https://sourceforge.net/projects/svarog86/

The boot sector template (`tools/dos/bootsector_1440.bin`) and the following
files in `tools/dos/image_root` were extracted from the Svarog86 boot image:

- `KERNEL.SYS`
- `COMMAND.COM`
- `TE.COM`
- `MEM.EXE`
- `MODE.COM`
- `MORE.EXE`
- `TREE.COM`

Svarog86 is a minimal FreeDOS distribution for 8086/8088-class machines. Keep
this note with redistributed binaries.
